## Font sources

**Primary: Fontshare** (fontshare.com) — curated, high-quality fonts with pre-built pairings. Your prompt includes a random sample of fonts and pairings from their catalog. CSS embed URL pattern: `https://api.fontshare.com/v2/css?f[]={slug}@{weights}&display=swap`

**Secondary: Open Foundry** (open-foundry.com) — smaller curated collection of open-source typefaces. Browse via search.

**Fallback: Google Fonts** — the model already knows these well, which is exactly the problem. Prefer Fontshare and Open Foundry for more distinctive choices. Google Fonts is fine when a specific well-known font genuinely fits.
